稍微解释一下上面代码的含义：

  • page -> 起始下标为 1 (默认值也是为1)
  • as_wide -> 是否宽屏 【1: 宽屏, 0: 小屏】
  • high_quality -> 是否高清 【1: 高清(最高1080p) / 0: 最低视频质量(默认)】
  • danmaku -> 是否开启弹幕 【1: 开启(默认), 0: 关闭】
  • allowfullscreen -> allowfullscreen= “ture” 允许全屏，使用该参数可以在浏览器中全屏播放
